⚠
Upsell structure is currently unstructured in HubSpot. The deals below were identified from dealtype = Existing Business set manually on some deals. Going forward, upsells will be owned by the Customer Success team using a dedicated pipeline and ownership model. See the CS Structure tab for the recommended setup.
How we identified the upsells
Since no dedicated upsell structure exists in HubSpot today, identification was done by cross-referencing three criteria:
Criterion 1
dealtype = Existing Business
The dealtype field already exists in HubSpot with options "New Business" and "Existing Business". Deals marked as Existing Business were the starting point of the analysis.
Criterion 2
Deal closed after AE start date
We verified when Sina Zarei (Jun 2025) and Denzel Kereama (May 2025) were added to HubSpot. Deals with dealtype = existingbusiness and closedate after that date were treated as active AE upsells.
Criterion 3
Package upgrade confirmed
For identified deals, we cross-referenced the current product_package against the deal history. All deals below show a transition from Standard to Advanced — confirming they are upgrades of existing customers, not new acquisitions.
Current limitation: this identification was done manually by cross-referencing data. There is no automation or dedicated pipeline. Some upsells may have gone untagged as existingbusiness. The CS structure described in the CS Structure tab resolves this permanently.

What makes these CS deals — the pattern
Signal 1 — Naming
"Upsell" / "(ADV/Upsell)" in deal name
Almost all are Standard → Advanced package upgrades on an existing account, tagged only via the deal name convention.
Signal 2 — Fields
dealtype = Existing Business · commission_type = Upsell
Set on roughly a third, blank on the rest — the name convention is currently more reliable than the fields.
The gap
customer_success_owner is empty on 100% of them
CS attribution today lives only in the spreadsheet. The "Deal | Set CS Owner from Company" workflow + backfill closes this gap.
Two profiles: package upsells on existing accounts, and CS-sourced net-new logos (intros/referrals — e.g. KSN, Haworth Tompkins, Studio CO2) that look like normal new business in HubSpot.
